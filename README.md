# Aviation Network Monitoring Dashboard

This project is a senior capstone mockup website for an **Aviation Network Monitoring Dashboard**.  
It is designed to demonstrate what a real-world aviation-related network operations dashboard could look like, including:

- network status overview
- latency and packet loss trends
- active alerts
- monitored node status
- service traffic visualization
- operational summary cards

The site is built with **React**, **Vite**, **Tailwind CSS**, **Recharts**, and **Lucide React**.

## Live Site

If GitHub Pages is configured correctly, the site will be available at:

[https://kessinger6.github.io/Aviation.network.dashboard.io/](https://kessinger6.github.io/Aviation.network.dashboard.io/)

## Project Purpose

This dashboard is a visual prototype created for an academic capstone project.  
It uses simulated data only and does **not** display live FAA or operational aviation network data.

## Features

- Aviation-style dashboard layout
- KPI cards for system health
- Interactive charts for latency, packet loss, and throughput
- Alert panel for simulated events
- Searchable monitored node table
- Mock operational summary for presentation/demo purposes

## Tech Stack

- React
- Vite
- Tailwind CSS
- Recharts
- Lucide React
- Framer Motion
